# UI升级说明 - 科技风格版本

## 🎨 设计升级

### 主页 (已完成)

#### 视觉效果
1. **动态渐变背景**
   - 5色渐变动画（紫色→粉色→蓝色→青色）
   - 15秒循环动画，营造科技感氛围
   - 浮动装饰球体，增加层次感

2. **毛玻璃效果 (Glass Morphism)**
   - 半透明卡片背景
   - 10px 背景模糊
   - 白色边框增强立体感

3. **交互动画**
   - 悬停时卡片上浮8px并放大
   - 图标缩放动画
   - 渐变光效过渡
   - 发光效果 (Glow)

4. **现代配色**
   - 蓝色系：法务智能体
   - 绿色系：财务智能体
   - 紫色系：人力智能体
   - 橙色系：IR智能体

#### 响应式设计
- **移动端** (< 768px)
  - 单列布局
  - 较小的padding和字体
  - 触摸友好的按钮尺寸

- **平板** (768px - 1024px)
  - 2列网格布局
  - 中等尺寸元素

- **桌面** (> 1024px)
  - 4列网格布局
  - 完整视觉效果

### 智能体页面 (已完成)

#### 视觉效果
1. **深色科技主题**
   - 深色渐变背景动画（#0f172a → #334155）
   - 20秒循环动画，营造专业科技感
   - 与主页风格统一但更加沉稳

2. **毛玻璃效果 (Glass Morphism)**
   - 侧边栏和配置面板使用深色毛玻璃
   - 消息气泡使用半透明玻璃效果
   - 输入区域和附件预览使用玻璃效果

3. **交互动画**
   - 消息淡入动画（fadeIn 0.3s）
   - 会话项悬停效果
   - 按钮渐变和阴影效果
   - 平滑的颜色过渡

4. **渐变配色**
   - 用户消息：渐变气泡（from-${color}-500 to-${color}-600）
   - AI消息：毛玻璃效果
   - 按钮：渐变背景 + 发光阴影
   - 标题：渐变文字效果

#### 移动端优化
1. **响应式布局**
   - 桌面端：3列布局（会话列表 + 配置面板 + 聊天区）
   - 移动端：单列布局 + 抽屉式侧边栏
   - 配置面板在移动端隐藏

2. **移动端侧边栏**
   - 汉堡菜单按钮（左上角）
   - 滑动抽屉动画
   - 半透明遮罩层
   - 点击遮罩或切换会话自动关闭

3. **触摸优化**
   - 更大的点击区域
   - 平滑的滑动动画
   - 适配手指操作的按钮尺寸

#### Markdown样式
- 链接：${color}-400 颜色 + 下划线
- 代码块：黑色半透明背景
- 行内代码：黑色半透明背景 + 圆角
